# ATM-Machine-Code
This code has the following functionalities:
  - Allows the user to log in to their ATM bank account with a limit of 3 trials
  - Gives the user the option to check their balance, withdraw, and deposit
  - If the user selects B for balance, their account balance is displayed
  - If the user selects W for withdraw, the amount they enter will be withdrawn from the amount available
  - If the user selects D for deposit, the amount they enter will be added to the initial balance.
